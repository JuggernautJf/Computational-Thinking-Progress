<details>
<summary><strong>First Time Setup (Windows cmd)</strong></summary>

### Assuming you are using cmd in Windows
- Install Git (if you don't have it).
- Clone this repository:
```cmd
git clone https://github.com/JuggernautJf/Computational-Thinking-Progress.git
```
- Install Python (if you don't have it).
- Create a virtual environment:
```cmd
python -m venv venv
```
- Run the virtual environment (assuming Windows):
```cmd
venv\Scripts\activate
```
- Install dependencies:
```cmd
pip install -r requirements.txt
```
